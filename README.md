## Procedure

### Data Cleaning and Preparation
In the initial data preparation phase, the following tasks were performed in Power Query:
- Data loading and inspection
- Deleting empty and unnecessary rows
- Adjusting column formatting

### Exploratory Data Analysis
This phase involved exploring the dataset with the aim of answering the questions below:
1. What is the total number of participants?
2. What is the average age of participants?
3. Which countries are the participants originally from?
4. What is the average salary by job title?
5. Which programming language do participants prefer?
6. How difficult was it for the participants to break into the data field?
